By: Bassel Saleh

------------------------------------------------
Overview:
This is a multi-page interactive personal website created as part of the WP1 course project. It uses HTML5, CSS3, Bootstrap 5, JavaScript, and jQuery. The website is fully responsive and runs offline in any browser.

Page Descriptions:

1. index.html (Home Page)
- Bootstrap navbar linking to all pages
- Welcome message with image using Bootstrap grid
- Responsive carousel with 3 rotating images

2. about.html
- Sections: biography, hobbies, personal goals
- "Show More" button reveals hidden content using JavaScript
- Light/Dark mode toggle using JavaScript
- Styled with external CSS

3. contact.html
- HTML5 form with fields: name, email, age, gender (radio), favorite language (select), message
- Includes required validation attributes
- Uses Bootstrap form styling

4. portfolio.html
- Displays 3 Bootstrap cards for sample projects
- One card updates dynamically using JavaScript on button click

5. fun.html
- jQuery used for:
   - slideToggle() effect on a hidden box
   - fadeToggle() on an image
   - AJAX load of quote.txt into a div using .load()

6. smart.html
- Input field with live personalized greeting (keyup)
- Background color change on hover using jQuery
- Animation (slide up/down) on click
- Includes a Light/Dark mode toggle

------------------------------------------------
Instructions:
- Open index.html in any browser to start.
- Ensure all files and images are kept in the correct folder.
- For full AJAX functionality (fun.html), use Live Server or local web server.
